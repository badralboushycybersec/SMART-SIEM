# Testing Datasets

This folder contains the final testing datasets used for evaluation.

## Notes

- Files are typically named like `final_testing_dataset_*`.
- Testing datasets in this repo are intended to include context/history features but do not apply SMOTE (per the dataset notes in [datasets](file:///d:/SIEM%20Paper/SMART-SIEM/datasets)).
- Different files correspond to different history depths (e.g., number of previous records included as context).
