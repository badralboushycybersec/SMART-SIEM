# Training Datasets

This folder contains the final training datasets used for model training.

## Notes

- Files are typically named like `final_training_dataset_*`.
- Training datasets in this repo are intended to include context/history features and (per the dataset notes in [datasets](file:///d:/SIEM%20Paper/SMART-SIEM/datasets)) use SMOTE for class imbalance handling.
- Different files correspond to different history depths (e.g., number of previous records included as context).
