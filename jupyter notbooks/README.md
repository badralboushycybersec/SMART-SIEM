# Jupyter Notebooks

This folder contains the notebooks used to create datasets, run experiments, and generate figures for SMART-SIEM.

## What You’ll Find Here

- Experiment notebooks (e.g., `smart_siem_v*.ipynb`) that explore different model variants and history depths.
- Dataset preparation notebooks (e.g., `create_Dataset_with_history.ipynb`, `handle_imbalance_dataset_SMOTE.ipynb`).
- Wazuh-focused notebooks (e.g., `wazuh_ai_knowledge_base.ipynb`, `wazuh_ai_self_adaptive.ipynb`).
- `ablation_results_v2.json`: saved results used for analysis/plots.
